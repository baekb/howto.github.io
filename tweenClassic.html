Classic Tween

Using the symbol, various animateions such as position movement,
size change, rotation, color effect, acclereation are implemented



Step 1
Converting an object into a symbol,
then set classic tween
Step 2
Insert the keyframe → dots are changed to pointer
select frame bar → change the tweening setting ex.direction of rotation, softeness, ...
Step 3
Select the key frame → change the setting of object ex.size, color effect, ...
Step 4
Set the last frame to keyframe
